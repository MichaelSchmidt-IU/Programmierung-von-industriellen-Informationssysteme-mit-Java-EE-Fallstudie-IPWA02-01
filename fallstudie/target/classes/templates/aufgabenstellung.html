1.1. Aufgabenstellung – Like Hero To Zero

Du arbeitest im Web-Development-Team einer PR-Agentur, die verschiedene Naturverbände und Vereine unterstützt. Ein Großspender hat Geld für eine App bereitgestellt, mit dem Ziel, eine Web-Frontend für ein Nachhaltigkeitsprojekt mit dem Namen „Like Hero To Zero“ zu entwickeln.

Die Anwendung soll:

  • öffentlich zugängliche Daten zu weltweiten CO₂-Emissionen anzeigen (ohne Login)
  • eine Backend-Oberfläche für Wissenschaftler:innen (mit Login) bieten

📋 User Stories (MoSCoW-Priorisierung)

  • MUST: Bürger:innen können CO₂-Ausstoß ihres Landes einsehen
  • MUST: Forschende können neue Klimadaten hinterlegen
  • COULD: Änderungen müssen ggf. erst freigegeben werden

🚧 Aufgaben

  1. Erstelle ein öffentliches GitHub-Repository für das Projekt.
  2. Entwickle einen Prototyp mit den MUST-Funktionen inkl. persistenter Datenhaltung mit:
    • JSF + CDI/Beans oder Spring Boot + Beans
    • JPA (z. B. Hibernate)
    • MySQL als relationale Datenbank
  3. Dokumentiere die Entwicklung in einem Fallstudienbericht:
    • Design
    • Screenshots
    • Datenbankstruktur
    • UML-Diagramm inkl. verwendeter Beans
